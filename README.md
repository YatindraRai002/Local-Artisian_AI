# 🎨 Kala-Kaart: AI-Powered Traditional Artisan Discovery Platform

> **Bridging India’s Cultural Heritage with Modern Technology**

Kala-Kaart is an AI-powered platform designed to connect traditional Indian artisans with the modern world. By combining machine learning, natural language processing, and multilingual chatbot technology, we make it easy to discover, connect with, and support India’s diverse craftspeople across regions and craft forms.

---

## 🌟 Vision

India’s traditional crafts represent centuries of heritage, yet many artisans remain unseen and struggle to access wider markets. Kala-Kaart exists to:

- **Preserve Heritage**: Digitally document and showcase India’s diverse crafts.
- **Empower Artisans**: Provide a global digital platform for artisans.
- **Break Language Barriers**: Offer inclusive multilingual support.
- **Enable Smart Discovery**: Use AI to connect customers with the right artisans.
- **Foster Livelihoods**: Create sustainable economic opportunities for rural communities.

---

## 🚀 Key Features

### 🤖 Multilingual AI Chatbot
- Supports **English, Hindi, Tamil, Telugu**
- Context-aware NLP with semantic search
- Conversation memory for smooth interactions

### 🔍 Intelligent Search & Discovery
- **Craft-based** and **location-based** filtering
- Search by state, district, or village
- AI-driven semantic matching for related terms

### 📊 Rich Artisan Database
- **1,000+ verified artisans** across **25+ states**
- **50+ craft types**: pottery, textiles, jewelry, woodwork, and more
- Real-time statistics and insights

### 🎯 Personalized Recommendations
- Tailored suggestions based on preferences
- Regional specialties and trending crafts
- “Similar Artists” discovery for related skills

## 🛠️ Setup & Installation

### Prerequisites
- Node.js 18+
- Python 3.8+
- npm or yarn

### Steps
1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd Local-Artisian_AI
````

2. **Install Dependencies**

   ```bash
   npm install
   pip install pandas numpy scikit-learn flask
   ```
3. **Data Setup**

   * Place `Artisans.csv` in the `public/` directory.
4. **Configure Environment**

   * Add `.env` file for Firebase and API keys.
5. **Run Development Servers**

   ```bash
   npm run dev
   cd backend && python app.py
   ```
6. **Build for Production**

   ```bash
   npm run build
   npm run preview
   ```
